# 🎮 Rock Paper Scissors Pro (Python GUI)

A modern and interactive Rock-Paper-Scissors game built using **Python (Tkinter)**.  
This project focuses on **clean UI, smooth user experience, and interactive gameplay** rather than just basic functionality.

---

## 🚀 Features

### 🎯 Gameplay
- User vs Computer game
- Best of 5 rounds system
- Real-time score tracking
- Clear round progression

### 🎨 UI/UX Improvements
- Dark-themed modern interface
- Dynamic instruction:
  - *"Select your move for Round X"*
- Highlighted action guidance
- Animated result (flash effect)
- Progress bar for round tracking

### 🖱️ Interactivity
- Button-based gameplay (no typing required)
- Keyboard shortcuts:
  - `R` → Rock  
  - `P` → Paper  
  - `S` → Scissors  
- Hover effects on buttons

### 🔊 Feedback System
- Sound feedback on each round (Windows only)
- Visual feedback using animations

---

## 🧠 Concepts Used

- Python functions & state management
- Event-driven programming (Tkinter)
- GUI design principles (UX flow & hierarchy)
- Dictionary-based logic
- Keyboard event binding
- Animation using `after()` method

---



## 🛠️ Technologies Used

- Python 3
- Tkinter (GUI)
- Random module
- Winsound (for sound effects)
