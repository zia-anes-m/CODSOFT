# 📝 GUI To-Do Manager (Python)

A feature-rich To-Do List application built using Python and Tkinter.  
This application helps users manage daily tasks with an interactive graphical interface and persistent storage.


## 🚀 Features

- Add, edit, and delete tasks  
- Mark tasks as completed  
- Priority management (High / Medium / Low)  
- Search and filter tasks  
- Sort tasks by priority  
- Reorder tasks (Move Up / Down)  
- Persistent storage using JSON  


## 🛠 Tech Stack

- Python  
- Tkinter (GUI)  
- JSON (Data Storage)  


## ▶️ How to Run

1. Clone the repository:
git clone https://github.com/zia-anes-m/CODSOFT.git

2. Go to folder:
cd CODSOFT/todo_list

3. Run:
python task_manager.py
