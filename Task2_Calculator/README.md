# 🧮 Advanced Calculator (Python GUI)

A clean and user-friendly calculator built using Python and Tkinter as part of my internship at CodSoft.  
This project focuses on functionality, usability, and a modern dark-themed interface.

---

## 🚀 Features

- ➕ Basic operations: Addition, Subtraction, Multiplication, Division  
- 📊 Percentage calculations (real calculator logic)  
- 🧠 Supports brackets `( )` for complex expressions  
- ✖️ Uses proper symbols (`×`, `÷`) for better UI  
- ⌫ Backspace and Clear functionality  
- ⌨️ Keyboard input support  
- 📜 History feature (view previous calculations)  
- 🌙 Clean dark mode interface  

---

## 🖥️ Tech Stack

- Python  
- Tkinter (GUI library)
